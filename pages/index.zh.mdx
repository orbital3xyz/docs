# 介绍
 
Orbital 为用户提供了一个无缝的 web3 钱包来访问区块链网络，并为开发者提供了一个强大的 web3 基础设施堆栈，使其能够轻松建立 web3 应用程序和游戏。

通过托管账户和部分密钥，简化来自 web2 的开发者和用户的使用门槛，并提供更复杂的 web3 本身不具备的能力，打破不可能三角困境（去中心化、安全和可扩展性）。

Orbital 被设计成一个模块化的 web3 系统，专注于开放标准、兼容性和开发者工具的互操作性。使用 Orbital，你可以只使用其中一个模块，也可以使用整个栈。同样，你也可以自由地将 Orbital 与你最喜欢的开发者工具结合起来。这符合 "web3 "的特征 -- 开放标准和兼容性 :)
